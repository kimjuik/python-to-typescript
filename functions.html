함수 (Functions)

Python과 TypeScript에서 함수 정의와 사용 방식의 차이를 알아보세요.

Python vs TypeScript 비교

Python

# Python 함수 정의
def greet(name):
    return f"Hello, {name}!"

# 함수 호출
result = greet("Alice")
print(result)  # Hello, Alice!

# 기본 매개변수
def greet_with_time(name, greeting="Good morning"):
    return f"{greeting}, {name}!"

print(greet_with_time("Bob"))  # Good morning, Bob!
print(greet_with_time("Charlie", "Good evening"))  # Good evening, Charlie!

TypeScript

// TypeScript 함수 정의
function greet(name: string): string {
    return `Hello, ${name}!`;
}

// 함수 호출
const result = greet("Alice");
console.log(result); // Hello, Alice!

// 기본 매개변수
function greetWithTime(name: string, greeting: string = "Good morning"): string {
    return `${greeting}, ${name}!`;
}

console.log(greetWithTime("Bob")); // Good morning, Bob!
console.log(greetWithTime("Charlie", "Good evening")); // Good evening, Charlie!
