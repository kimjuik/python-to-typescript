클래스 (Classes)

Python과 TypeScript 모두 객체 지향 프로그래밍을 지원하며 클래스를 사용해 객체를 정의할 수 있습니다. 그러나 문법과 타입 시스템에서 차이가 있습니다.

Python vs TypeScript 비교

Python

# Python 클래스 정의
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def greet(self):
        return f"안녕하세요, 제 이름은 {self.name}이고 {self.age}살입니다."

# 클래스 상속
class Employee(Person):
    def __init__(self, name, age, employee_id):
        super().__init__(name, age)
        self.employee_id = employee_id
    
    def greet(self):
        base_greeting = super().greet()
        return f"{base_greeting} 직원 ID는 {self.employee_id}입니다."

# 인스턴스 생성 및 메서드 호출
john = Employee("John", 30, "E12345")
print(john.greet())

TypeScript

// TypeScript 클래스 정의
class Person {
    name: string;
    age: number;
    
    constructor(name: string, age: number) {
        this.name = name;
        this.age = age;
    }
    
    greet(): string {
        return `안녕하세요, 제 이름은 ${this.name}이고 ${this.age}살입니다.`;
    }
}

// 클래스 상속
class Employee extends Person {
    employeeId: string;
    
    constructor(name: string, age: number, employeeId: string) {
        super(name, age);
        this.employeeId = employeeId;
    }
    
    greet(): string {
        const baseGreeting = super.greet();
        return `${baseGreeting} 직원 ID는 ${this.employeeId}입니다.`;
    }
}

// 인스턴스 생성 및 메서드 호출
const john = new Employee("John", 30, "E12345");
console.log(john.greet());
