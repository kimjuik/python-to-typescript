문제 2: 선택적 매개변수

다음 함수에 타입을 추가하고, 두 번째 매개변수를 선택적으로 만드세요. 두 번째 매개변수가 전달되지 않으면 "world"를 기본값으로 사용합니다.

function greet(name, greeting) {
    if (greeting) {
        return `${greeting}, ${name}!`;
    } else {
        return `Hello, ${name}!`;
    }
}
