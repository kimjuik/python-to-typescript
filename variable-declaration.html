변수 선언 (Variable Declaration)

Python과 TypeScript에서 변수 선언 방식의 차이를 알아보세요.

Python vs TypeScript 비교

Python

# Python에서는 타입을 명시하지 않음
x = 10
name = "John"

# 변수 타입은 실행 중에 동적으로 결정됨
x = "Hello"  # 문제 없음

TypeScript

// TypeScript에서는 타입을 명시하거나 추론 가능
let x: number = 10;
let name: string = "John";

// 타입이 고정되어 있음
x = "Hello"; // 오류 발생!
